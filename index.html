Regroupement de logs

Le code de ce projet est réalisé en PHP. Il permet de récupérer toutes les logs d'un dossier, en utilisant principalement les expressions régulières (preg_match) pour identifier des morceaux de log spécifiques tels que "ERROR", ainsi que les dates et les messages.

Le processus commence par la lecture du dossier pour parcourir tous les fichiers avec l'extension ".log". Le chemin vers le dossier est indiqué dans le code. Les dates ont un format variable d'une application à l'autre, mais le programme les détecte toutes et les modifie pour les normaliser dans un format commun.

Les messages de log sont extraits et découpés en parties distinctes. Le type est facilement identifiable en recherchant les termes "ERROR" ou "UNEXPECTED". L'objectif est de découper chaque ligne pour éliminer les caractères spéciaux souvent superflus, propres à chaque application, puis d'afficher les logs dans une page intéractive.
